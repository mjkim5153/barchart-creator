# 운항스케줄 바차트 표출 규칙

## 개요

`process_flight_schedule()`로 수집된 데이터를 기반으로 바차트 영역에 운항스케줄을 표출할 때 적용되는 규칙을 정의합니다.
데이터 컬럼 구조는 `ubkiais_rule.md` 참고.

---

## 1. 데이터 필터링

바차트에 표출하기 전 아래 조건으로 데이터를 필터링합니다.

| 조건 | 설명 |
|------|------|
| `Status != 'CNL'` | 결항편 제외 |
| `Bt_idx == 'Y'` | 중복 레코드 제거 (Y인 레코드만 표출) |

---

## 2. 항공기별 편 연결 규칙

- 같은 기번(`Acno`)에 속한 항공편을 **`STD_KST` 오름차순**으로 정렬하여 순서대로 연결
- 연결된 편들은 바차트에서 시간 흐름에 따라 이어서 표출

### 연결 무결성 검증

- **규칙**: 이전 편의 도착공항(`Arrstn`) = 다음 편의 출발공항(`Depstn`) 이어야 함
- **위반 시**: 해당 바차트에 **빨간색 테두리** 표시

```
[정상 연결]
편 A: ICN → GMP  →  편 B: GMP → PUS  ✓ (GMP == GMP)

[연결 오류 - 빨간 테두리]
편 A: ICN → GMP  →  편 B: PUS → ICN  ✗ (GMP ≠ PUS)
```

---

## 3. 바차트 모양 및 표출 정보

### 바차트 형태

- 직사각형 형태
- BAR 구간: `STD_KST` ~ `STA_KST`

### 바차트 내부 표출 정보

```
    ┌──────────────────────────────┐
    │           KE123              │
    └──────────────────────────────┘
ICN                                 GMP
```

| 위치 | 표출 내용 |
|------|-----------|
| BAR 내부 중앙 | 편명 (`Fltno`) |
| BAR 하단 좌측 바깥 | 출발공항 (`Depstn`) |
| BAR 하단 우측 바깥 | 도착공항 (`Arrstn`) |

---

## 4. 바차트 배경색

| 조건 | 배경색 |
|------|--------|
| `Depstn == 'GMP'` 또는 `Arrstn == 'GMP'` | 파란색 |
| 그 외 | 주황색 |

---

## 5. 바차트 사이 GROUND TIME 표시

- 연결된 두 바차트 사이의 **정 가운데**에 GROUND TIME을 텍스트로 표시
- GROUND TIME = 다음 편 `STD_KST` - 이전 편 `STA_KST`

```
예시:
│ KE123 ICN→GMP │  45분  │ KE456 GMP→PUS │
         STA_KST        STD_KST
```

> **참고**: "GT:" 접두어 없이 숫자+분만 표시

---

## 5-1. 교대 행 배경색

- 기번 행의 배경색을 교대로 적용하여 가독성 향상
- 짝수행: transparent (투명)
- 홀수행: #d9d9d9 (연한 회색)
- **Y축 라벨 영역에도 동일한 교대 배경색 적용** (차트 영역과 일관성 유지)

---

## 5-2. Y축 넘버링

- Y축 라벨에 순번을 표시: `1 HL7234`, `2 HL7235` 형식
- MARGIN.left를 110px로 확장하여 번호 공간 확보

---

## 5-3. 줌/스크롤 동작

| 입력 | 동작 |
|------|------|
| 마우스 휠 | Y축 스크롤 (세로 이동) |
| Ctrl+휠 / 핀치 | X, Y 동시 줌 |

### X축 범위 제한
- X축 범위는 검색 조건(시작일/종료일)이 아니라 **실제로 표출되는 항공편의 STD_KST/STA_KST 최소~최대값**으로 결정됨 (`static/js/barchart.js`에서 데이터 기반으로 산출)
- 단일일 검색이라도 ARR 도착편의 실제 출발(STD)이 검색일 전날 저녁인 야간편(예: 장거리 국제선)은 그 전날 저녁부터 바가 시작되도록 X축이 자연 확장됨 — 검색일 00:00~24:00로 강제로 압축하지 않음
- 다일 시 각 날짜의 00:00 위치에 "M/D" 형식 날짜 라벨 표시
- 날짜 경계선은 파란색 점선으로 표시

### 방향키 스크롤
- ArrowLeft/Right: 수평 스크롤

### 키보드 줌
- Ctrl+ArrowRight: X축 확대
- Ctrl+ArrowLeft: X축 축소
- Ctrl+ArrowUp: Y축 확대
- Ctrl+ArrowDown: Y축 축소

### 축소 시 텍스트 숨김
- ROW_HEIGHT * yZoom < 30px이면 BAR 내부 텍스트(편명, 공항, GT) 숨김
- 마우스 오버 툴팁은 축소 상태에서도 항상 표시

---

## 6. 마우스 오버 툴팁

바차트에 마우스 오버 시 아래 정보를 툴팁으로 표시합니다.

| 항목 | 컬럼 |
|------|------|
| 편명 | `Fltno` |
| 기번 | `Acno` |
| 출발공항 | `Depstn` |
| 도착공항 | `Arrstn` |
| 출발 예정시각 | `STD_KST` |
| 도착 예정시각 | `STA_KST` |
| 블록타임 | `Blocktime` (분) |
| 그라운드타임 | 이전 편 `STA_KST` - 해당 편 `STD_KST` (분) |

> **GROUND TIME 계산 방향**:
> - 해당 편 기준으로 **앞에 연결된 도착편의 `STA_KST`** - **해당 편의 `STD_KST`**
> - 첫 번째 편(앞에 연결된 편 없음)은 GROUND TIME 미표시

---

## 규칙 요약

```
[데이터 필터] Status != CNL  AND  Bt_idx == Y
      ↓
[정렬] 기번(Acno) 별로 STD_KST 오름차순
      ↓
[연결 검증] 이전 Arrstn == 다음 Depstn
           → 불일치 시 빨간 테두리
      ↓
[배경색] GMP 포함 시 파란색 / 그 외 주황색
      ↓
[BAR 표출] 출발공항 | 편명 | 도착공항
      ↓
[사이 텍스트] GROUND TIME (분)
      ↓
[마우스 오버] 편명, 기번, 출도착공항, STD/STA_KST, BT, GT
```
