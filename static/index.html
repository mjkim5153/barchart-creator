정시율 분석(시나리오)

    시나리오

    비운항 처리할 편명을 선택하세요. 현재 로드된 전체 기간의 모든 날짜에서 해당 편명의 기번(Acno)/D_OPER가 빈칸으로 바뀌어 바차트 상단 CANCEL 행으로 빠지고, 정시율 집계에서도 자동 제외됩니다. 특정 날짜에 그 편명이 없으면 (이미 결항/비운항) 조용히 건너뜁니다.

    한 줄에 하나씩 "앞편-뒤편" 형식으로 입력하세요. (예: 7C106-7C117)
    앞편의 기번은 고정되고, 뒤편과 뒤편이 속한 기번의 그날 운항 체인에서 뒤편 이후 모든 편이 앞편의 기번으로 변경됩니다. 현재 로드된 전체 기간의 모든 날짜에 적용을 시도하며, 앞/뒤편 중 하나라도 없는 날짜는 건너뜁니다 (비운항 처리된 편은 "없음"으로 취급됩니다 — 비운항 시나리오를 먼저 적용하세요).

    저장/불러오기

    현재 바차트/시나리오 조작 상태에 이름을 붙여 저장합니다. 나중에 "불러오기" 탭에서 다시 불러와 이어서 작업할 수 있습니다. 정시율 분석(실적)은 저장/불러오기와 무관하게 항상 최초 업로드/크롤링 시점의 원본 데이터를 기준으로 유지됩니다.

    저장된 목록에서 불러올 항목을 선택하세요. 불러오면 현재 화면의 바차트/시나리오 상태가 저장 시점으로 교체됩니다.
